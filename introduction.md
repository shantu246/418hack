# PING —— 跨越时空的地理坐标留言册

**“每一次跳动，都是一个 ping。”**

## 💡 产品理念 (Philosophy)
在这个信息过载的数字时代，我们的记忆被轻易地淹没在无尽的瀑布流中。**PING 致力于让数字记忆重新拥有“物理重量”。** 

我们相信，最真实的情感往往与特定的空间紧密绑定。PING 拒绝虚拟世界里的漫无目的，它是一款基于地理位置的半匿名社交记录应用。它鼓励用户走出信息茧房，用双脚去丈量真实的世界。在这里，所有的心情、故事和秘密都被锚定在具体的经纬度上。当你来到某个坐标，你不仅抵达了一个物理地点，更走进了一段跨越时空的记忆。通过半匿名的机制，我们摒弃了容貌焦虑与社交压力，只留下最纯粹的、基于“同一片天空下”的情感共鸣。

## 🚀 使用方法 (How it Works)
PING 的核心玩法围绕“探索-留下-解锁”展开，打造一种轻量级、带有寻宝感的现实交互体验：

1. **Drop a ping (留下印记)**：当你抵达某个令你触动的地点，掏出手机发送一个 ping。你可以选择“私语(仅自己可见)”、“幻影(阅后即焚)”或“经典(一段时间保留)”，用文字、即时拍摄的照片或视频，将此刻定格在当前坐标。
2. **Radar Mode (雷达寻索)**：打开 App，地图不会直接显示精确的图钉，而是以模糊的热力图展示周围 500 米内隐藏的 pings。你只能感知到这片区域有故事发生，但必须亲自前往探索。
3. **Open a ping (解锁时空)**：当你跟随雷达指引，物理距离靠近某个 ping 不足 50 米时，它才会在地图上具象化。手动点击即可“Open”，阅读他人（或曾经的自己）留下的心声，并进行轻量的匿名互动。
4. **AI Journey (月度回望)**：每个月末，AI 将根据你本月解锁和留下的 pings，智能提炼你的情绪关键词（如“平静”、“惊喜”），并生成一份专属的“足迹与情感月报”，带你重温这个月走过的街角与心境。

## 🌟 体验与场景 (Scenarios)

* **📍 发现被忽视的角落**
生活不只有宿舍、教学楼和公司的两点一线。也许是校园里那条鲜有人问津的林荫小道，也许是城市边缘一家没有招牌的旧书店。ping 鼓励你偏离日常的既定路线，去探索那些平时被**忽视的角落**。当你在地图的“未知区域”点亮一个 ping，你便赋予了那个平凡之地独一无二的意义。

* **📷 标记你爱好的事物**
无论是在街角偶遇并喂食的一只橘猫，还是雨后天台上一场转瞬即逝的晚霞。你可以用文字、阅后即焚的照片或一段短视频，把这些你所**爱好的事物**，精准地“锚定”在它们发生的地方。这不仅是地理位置的打卡，更是你情绪与热爱的专属坐标。

* **⏳ 寄给三年后的自己**
ping 不仅仅是连接陌生人的雷达，它更是一枚属于个人的时间胶囊。你可以选择在一个对你意义非凡的地点（比如毕业典礼的礼堂前、第一次独自旅行的车站），留下一个私密的 ping，作为一份**给三年后的自己**的礼物。当未来的某一天你再次回到原地，那个曾经的自己会在那里静静地等待着你，为你讲述当年的期许与故事。
